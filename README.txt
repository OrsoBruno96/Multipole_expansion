Questo programmino dovrebbe riuscire ad ottenere l'espansione in multipoli per un sistema di cariche puntiformi generico, da inserire all'interno del programma calcolo_sviluppo_cariche_puntiformi.py, dove c'è una mini-guida.

il programma scrivi_armoniche_grado_l.py semplicemente scrive su un file .tex tutte le armoniche di ordine l, se servisse per confronto.

calcolo_sviluppo_cariche_puntiformi_2.py è in fase di sviluppo, al momento non fa niente di più del primo

tutti i programmi hanno come output un file .tex che è da compilare per vedere l'out

per eseguire i file .py serve aver installato due pacchetti, sympy e mpmath
