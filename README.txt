Questo programmino dovrebbe riuscire ad ottenere l'espansione in multipoli per un sistema di cariche puntiformi generico, da inserire all'interno del programma calcolo_sviluppo.py, dove c'è una mini-guida.

il programma scrivi_armoniche.py semplicemente scrive su un file .tex tutte le armoniche di ordine l, se servisse per confronto.

calcolo_sviluppo_test.py è in fase di sviluppo, al momento non fa niente di più del primo, anzi, non funziona. Dovrebbe estendere in modo rapido al caso continuo il tutto

tutti i programmi hanno come output un file .tex che è da compilare per vedere l'out

per eseguire i file .py serve aver installato due pacchetti, sympy e mpmath
